Here's how to run your sweepstake:

  1. Share your code — send the 6-character code below to your players so they can join.
  2. Set up the draw — once you're in, go to the Draw tab, enter player names, then tap Start the draw. Teams are revealed one by one.
  3. Save & go live — after the draw completes, tap Save & go live and enter your PIN. All players instantly see the results.
  4. Reclaim admin anywhere — if you join on a different device, tap Settings → enter your PIN under Admin access to restore full control.
